# Nigeria Macro-Economic Stability Analysis (2002–2025)

This repository contains the quantitative models and data visualizations supporting the research paper: **"The Merchant Bank Paradox: Liquidity Decay and the Debt-Growth Nexus."**

## **Key Research Metrics**
* **Merchant Bank Paradox:** Modeled institutional correlation ($r = 0.9977$) during structural volatility clusters.
* **Liquidity Decay:** Quantified erosion of purchasing power with a calculated Beta coefficient ($\beta = -232,335.35$).
* **Labor Precariousness Index:** Analysis of 73.5% non-permanent staffing risk in the financial sector.
* **2025 Dashboard:** Predictive analysis of the $34.6\%$ inflation peak and $63.1\%$ debt-service ratio.

## **Methodology**
The analysis utilizes **Time-Series Regression**, **Pearson Correlation**, and the **Chow Test for Structural Breaks**. It bridges the 2016 liquidity crisis with the 2023–2025 fiscal shocks, identifying the "Paradox Zone" where traditional monetary transmission mechanisms fail.

## **Repository Structure**
* **/Data**: Contains the `cleaned_inflation_data.csv` used for the final models.
* **/Results**: High-resolution (.png) dashboards for policy review and journal submission.
* **Nigeria_Macro_Analysis.ipynb**: Full Jupyter Notebook containing the Python code, Chow Test, and Export scripts.
* **requirements.txt**: Environment setup for computational reproducibility.
