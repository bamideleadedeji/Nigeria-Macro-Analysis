# Nigeria Macro-Economic Stability Analysis (2002–2025)

This repository contains the quantitative models and data visualizations supporting my research on systemic risk and financial sector resilience in Nigeria.

## Key Research Metrics
- **Merchant Bank Paradox:** Modeled institutional correlation ($r > 0.99$).
- **Liquidity Decay:** Calculated Beta coefficient ($\beta = -232,335.35$).
- **Labor Precariousness Index:** 73.5% non-permanent staffing risk analysis.
- **2025 Dashboard:** Analysis of the 34.6% inflation peak and 63.1% debt-service ratio.

## Methodology
The analysis utilizes **Time-Series Regression** and **Pearson Correlation** to identify structural imperfections within the Nigerian banking sector. It bridges the 2016 liquidity crisis with the 2024-2025 fiscal shocks.

## Repository Structure
- `/scripts`: Python models for liquidity and volatility tracking.
- `/visuals`: High-resolution dashboards for policy review.
- `requirements.txt`: Environment setup for reproducibility.
